MEMORIA DESCRIPTIVA


Las elecciones de diseño que se tomaron para realizar la web estan relacionadas con un gusto mas bien personal, y se refleja una idea principal metaforica con el universo como algo infinito y en constante transormación, al igual que la posibilidad de crear y diseñar cada trabajo de diseño.

Se decidio utilizar imagenes de propia autoria para acompañar la navegación por la web junto con transiciones css3 e para dar un toque más dinamico, al igual que la botonera con imagenes por sustitución para que sea mas interactiva.
Los colores principales utilizados son el blanco (#FFFF) y negro (#0000), como color secundario un verde claro (#63DEBE).
